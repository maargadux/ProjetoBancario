# 🏦 Projeto Bancário em Java

Aplicação simples desenvolvida em Java com foco na prática de Programação Orientada a Objetos (POO).

O projeto simula uma conta bancária básica, permitindo operações como depósito, saque e consulta de saldo.

---

## 🚀 Funcionalidades

- 💰 Depositar valores
- 💸 Realizar saques
- 📊 Consultar saldo atual

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Criação de classes e construtores
- Manipulação de atributos
- Métodos e regras de negócio
- Encapsulamento (em evolução)
